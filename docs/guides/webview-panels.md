# Interactive WebView Panels Guide

## Diff Preview Panel

- Visualize proposed code changes before applying refactor or migration.
- Highlight added, removed, and modified lines.
- One-click rollback or apply changes.

## Dependency Graph Panel

- Visualize project dependencies and symbol relationships.
- Identify circular dependencies and refactor opportunities.
- Interactive navigation between files and modules.

## Guided Walkthroughs

- Step-by-step migration and refactor flows.
- Inline documentation and best practices.
- UI feedback for test results and errors.

## Implementation Notes

- Use VSCode WebView API for custom panels.
- Integrate with core engine and adapters for live data.
- Document all UI features and user flows.
